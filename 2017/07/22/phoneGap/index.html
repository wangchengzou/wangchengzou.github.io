phoneGap基础知识

2017-07-22
App phoneGap

一、phoneGap概述

国内的hybridApp：appcan 369clound。。。
phonegap.com
build.phonegap.com
cordova.apache.org
what？是一个开源的框架，提供了丰富的插件（让调用原生应用程序才能实现的功能）
where？采用前端技术构建hybridApp
why？
核心特点：
兼容（核心理念：write once,run everywhere）
标准化（完全遵循w3c标准）
大众化（不需要专业的移动开发知识，只要会前端，就可以开发移动端的app）

历史：
08 iphoneDevCamp推出理念：使用web技术调用iOS平台接口
09、10 支持超过6个移动端的平台
11年 被ADobe收购，phoneGap依然保留商标权，将核心的跨平台代码贡献给了ADobe，ADOBE宣布将核心代码开源成新项目：cordova
12年 推出phonegap build服务，实现云端打包

how？
搭建环境：
    1、phoneGap官方建议
    2、cordova ：cli

二、搭建phonegap开发环境

1、安装pc端的应用程序
2、安装移动端的app
    https://github.com/phonegap/phonegap-app-developer/tree/master/resources/release/android
    ①找到apk格式结尾的文件，统一phoneGap_develop.apk,放到了c盘的根目录下
    ②启动模拟器
    ③将apk文件安装到模拟器中

    adb： Android Debug Bridge 安卓调试桥
    启动一个小的服务，可以连接到当前和pc连接的android设备，设备、包的管理

    安装：
    启动命令行 windows+R

3、在pc端管理phoneGap项目
    新建（联网验证版本的过程）或者
    打开已经存在的（template.zip）
    ①放到c盘根目录下
    ②解压缩到template文件夹中
    ③通过pc端的phoneGap打开template项目
    点击+号，选择打开已经存在的项目，从目录中找到c:/template/template，点击选择即可

4、在移动端的app预览pc效果

三、phoneGap

1、phoneGap所支持的移动端的事件
    事件：
    ①deviceReady
    ②运行状态 pause resume
    ③电池 batteryStatus ..
    ④按键 volumeDownButton...

    如果绑定这些事件：
    document.addEventListener()

2、phoneGap所提供的各种插件

①device插件
    device.name/platform/version..
    关于插件的工作原理：
    通过js调用插件中提供的对象或者方法属性时，phoneGap会根据你运行的当前的平台（android/iOS/wp。。。）
    找到原生的代码，执行数据的获取等操作，将结果返回给js。

②加速度传感器 accelerometer
    navigator.accelerometer.getCurrentAccelration/watchAccelration/clearWatch
    在学习插件的过程中，注意：支持的平台有哪些，指定的平台是否有特殊的情况去处理

③摄像头 camera
    navigator.camera.getPicture() //拍照或者从相册选择照片

练习：
准备工作：保存几张照片到模拟器
①getPicture 拍照还是选照片：sourceType
②拿到选择的照片的文件信息
④通知 notification
    navigator.notification.alert/confirm/beep/vibrate
⑤file 文件操作插件
    功能：在android的文件系统中创建一个文件test.txt,向文件写入'Hello PhoneGap';再从文件中内容读出来显示在弹窗中
    用到的对象
    写操作为例：
    fileSystem 文件系统对象
        // 请求当前持久化的文件系统
        window.requestFileSystem
    fileEntry 文件操作的入口对象
        //创建新的文件或查询一个存在文件
        getFile
    fileWriter 文件写对象
        //创建一个可以写入文件的写对象
        createWriter()
        //写入内容
        fileWriter.write()
    读操作为例
    fileSystem
    fileEntry
    fileReader
    //文件读对象
    通过new的方式：new FileReader();
    fileReader.readAsText()

    file 文件对象
    //FileEntry对象的file方法获得一个File对象实例
⑥capture 采集
    采集音频、视频、图片信息
    navigator.device.capture.captureAudio/Image/Video()
